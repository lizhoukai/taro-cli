## 安装项目依赖
```bash
cnpm i
```

## 安装脚手架用于编译生成 dist 文件
```bash
cnpm install -g @tarojs/cli
```

## 开发环境
```bash
npm run dev:weapp // 微信
npm run dev:alipay // 支付宝
```

## 测试环境
```bash
npm run test:weapp // 微信
npm run test:alipay // 支付宝
```

## 正式环境
```bash
npm run prod:weapp // 微信
npm run prod:alipay // 支付宝
```

## 代理接口
```bash
config 目录 => API_TARGET
```
